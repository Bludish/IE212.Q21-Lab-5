# Thư mục video mẫu

Đặt file video giả lập camera vào thư mục này với tên `sample.mp4`
(hoặc đổi tên trong biến môi trường `VIDEO_PATH` ở `docker-compose.yml`).

Gợi ý nguồn video test miễn phí (có người đi lại):
- https://www.pexels.com/search/videos/people%20walking/
- https://pixabay.com/videos/search/pedestrians/

Yêu cầu: định dạng `.mp4`, khuyến nghị độ phân giải vừa phải
(720p trở xuống) để giảm tải khi xử lý trên CPU.
